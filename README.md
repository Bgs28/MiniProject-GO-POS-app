# MiniProject-GO-POS-app

A lightweight and efficient Point of Sale (POS) system built with Go. This application manages basic retail operations including product inventory, transactions, and sales reporting.

# Features
Product Management: CRUD (Create, Read, Update, Delete) products with category and stock tracking.

Transaction System: Process sales, calculate totals, and manage shopping carts.

Inventory Updates: Automatic stock reduction upon successful transactions.

Simple Authentication: Secure access for admin/cashier roles.

Clean Architecture: Organized code structure for better maintainability.

# Tech Stack

Language: Go (Golang)

Database: MySQL
